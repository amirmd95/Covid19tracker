## 🚀 5-Step Quick Deploy (Option 1)

```
STEP 1: Push to GitHub
├─ git init
├─ git add .
├─ git commit -m "Initial commit"
└─ git push

STEP 2: Deploy Backend (Render)
├─ Sign up at render.com
├─ Connect GitHub repo
├─ Configure: backend folder, Java 17
└─ Get URL: https://YOUR-APP.onrender.com

STEP 3: Update Frontend
├─ Edit: environment.prod.ts
├─ Add backend URL
└─ git push

STEP 4: Deploy Frontend (Vercel)
├─ Sign up at vercel.com
├─ Import GitHub repo
├─ Configure: frontend folder, Angular
└─ Get URL: https://YOUR-APP.vercel.app

STEP 5: DONE! 🎉
└─ Share your link with anyone!
```

## 🆘 Troubleshooting

```
Problem: "Failed to load data"
Solution: Backend is sleeping, wait 60 seconds

Problem: "Build failed on Render"
Solution: Check Java 17 is selected

Problem: "Build failed on Vercel"
Solution: Check Node 18 is selected

Problem: "CORS error"
Solution: Already fixed! CORS accepts all origins
```
